Unit Selection

User can click to show/hide unit selection menu. Units are set using an apply/revert modality because multiple of them can be set at once. Changing the units resets everything. Units only work when using POST.

Substance Selection

User can select a substance via this menu. Changing the substance will completely reset all data. Air has some mild issues because igmix lags development on the backed.

Property Input

This is where the user can input property data, using whatever units have been apply'd. Only two properties can be specified at one time. Adjusts visible for mp substances vs. ideal gases. Button submits HTTP POST request.

Property Display Selection

User can specify which properties should be shown in the table. Expand and adjust the checkboxes as desired.

Isoline Display Selection

User can specify which isobars should be shown in the plot. Expand and adjust the checkboxes as desired.

Property Table

Most interactivity provided by Datatables. Should be responsive to which properties are checked in the property selection. Lots of cleanup and display optimization is needed here, but the basic functionality exists. Prettify!

Plot

The plot responds to the points added to the table. It should display all the points that were added. It should also allow the user to click to add a point for substances (subject to the points being in range). Hover tooltip works with extra property info. Lots of prettification needed here.
