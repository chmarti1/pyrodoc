Features of PYroMat

PYroMat provides thermodynamic properties for over 1,000 substances. Fast and intuitive access to the properties of each substance comes from calling up objects. Properties are calculated by object methods that support array-like objects as arguments.

>>> import pyromat as pm
>>> air = pm.get('ig.air')
>>> air.gam(T=[300, 400, 500, 600])
array([1.39990859, 1.39519249, 1.38666408, 1.37575277])

PYroMat has property models for ideal gases, ideal gas mixtures, and multi-phase (liquid/vapor) pure substances. The properties provided include

  • cp Isobaric specific heat
  • cv Isochoric specific heat
  • d Density
  • e Internal energy
  • gam Specific heat ratio
  • h Enthalpy
  • mw Molecular weight
  • R Ideal gas constant
  • s Entropy

PYroMat includes an info() function to help users find the substances they are looking for. Users can search for substances by their collection ("ig" for ideal gas, "mp" for multi-phase), their name (e.g. "H2O"), or their atomic contents (e.g., "H" and "C" for hydrocarbons).

Since the property models are assembled from a number of sources, the info() function can also be used to recall detailed documentation and cited sources for each substance. Once you have imported PYroMat, just type 

import pyromat as pm
help(pm.info)
for more information. 

>>> import pyromat as pm
>>> pm.info()
