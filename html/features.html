Features of PYroMat

PYroMat provides thermodynamic properties for nearly 1,000 substances. Fast and intuitive access to the properties of each substance comes from calling up objects. Properties are calculated by object methods that support array-like objects as arguments.

If you want to calculate the properties of air, you load PYroMat, then, you ask for PYroMat's air model. Then, you ask it whatever you want to know. 

>>> import pyromat as pm
>>> air = pm.get('ig.air')
>>> air.gam(T=[300, 400, 500, 600])
array([1.39990859, 1.39519249, 1.38666408, 1.37575277])

PYroMat has property models for ideal gases, ideal gas mixtures, and multi-phase (liquid/vapor) pure substances. The properties provided include

  • cp Isobaric specific heat
  • cv Isochoric specific heat
  • d Density
  • e Internal energy
  • gam Specific heat ratio
  • h Enthalpy
  • mw Molecular weight
  • R Ideal gas constant
  • s Entropy
  • v Specific volume
  • x Quality

As of version 2.2.0, PYroMat gives users lots of flexibility in how they define states. For example, many of these properties may be passed interchangeably as arguments 

>>> h2o = pm.get('mp.H2O')
>>> h = h2o.h(T=300,p=1.01325)
>>> h2o.T(h=h,p=1.01325)
array([299.99999892])
>>> h2o.s(T=300, p=1.01325)
array([0.39306625])
>>> h2o.s(h=h, p=1.01325)
array([0.39306624])

To calculate lots of properties all at once, it is far more efficient to use the new state method implemented in version 2.2.0. This approach returns a dictionary with just about all the properties a user could want. 

>>> h2o.state(T=300, p=1.01325)
{
    'p': array([1.01325]), 
    'T': array([300.]), 
    'd': array([996.55693478]), 
    'x': array([-1.]), 
    'e': array([112.55442276]), 
    'h': array([112.65609784]), 
    's': array([0.39306625]), 
    'cp': array([4.18068028]), 
    'cv': array([4.13021821]), 
    'gam': array([1.01221777]), 
    'v': array([0.00100345])
}

PYroMat includes the info() and search() functions to help users find the substances they need. Users can search for substances by their collection ("ig" for ideal gas, "mp" for multi-phase), their name (e.g. "H2O", "water", or "R-134a"), or their atomic contents (e.g., "H" and "C" for hydrocarbons).

Since the property models are assembled from a number of sources, the info() function can also be used to recall detailed documentation and cited sources for each substance. Once you have imported PYroMat, just type 

import pyromat as pm
help(pm.info)
help(pm.search)
for more information.

The PYroMat R134a multiphase data are adapted from [1]. The equation of
state used to calculate most properties is their original work, but the
saturation conditions are derived from Wagner [2].
The data are implemented with the MP1 class. In-line help describes the
available funcitons and their proper use.
Adaptation to PYroMat was by Chris Martin in June 2018 thanks, in part, to
a grant from Penn State's Open Education Resources Initiative.
