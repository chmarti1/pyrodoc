Download PYroMat

Here are some options for downloading and installing PYroMat.

Python package index
Github
Sourceforge

PYroMat supports both Python 2 and Python 3. If you download a version later than 1.3 from Github or Sourceforge, the root directory will also contain "test.py," which generates a .log file showing the results of tests of each of PYroMat's classes.

The Python package index

The PYroMat package index can be found at https://pypi.python.org/pypi/PYroMat. If you already have PIP installed, you can download and install PYroMat with a single command. To install from a command line, run the following command as an administrator:

pip install pyromat
To upgrade from an older version
pip install pyromat --upgrade

If you are not logged in as an administrator, but you want to be able to install PYroMat globally on your system, pip recommends 

    sudo -H pip install pyromat

Github

The PYroMat package index can be found at https://github.com/chmarti1/PYroMat. If you already have GIT installed, fire up your command line: 

    git clone https://github.com/chmarti1/PYroMat.git
    cd PYroMat
    python setup.py install

The last command might need to be executed by an administrator on your system. In Linux, this is usually best done with the command 

    sudo -H python setup.py install

The PYroMat issues page on Github is the authoritative record of all errata and their resolutions.

Sourceforge

The PYroMat sourceforge page can be found at https://sourceforge.net/projects/pyromat/. Choose one of the .zip or .gz files, download it, and extract it. In the root directory, you will find the distribution files includeing the setup.py file. Fire up your command line:
python setup.py install
